Project Title:
Sevalink Care: Digital Reliability Audit
Project Description:
This project focuses on testing the Sevalink Care website to ensure its functionality, usability, validation, and responsiveness across different devices and user inputs.
Objective:
To identify bugs, improve system reliability, and ensure a smooth user experience through structured testing techniques.
Modules Covered:
Functional Testing
Validation Testing (BVA)
Usability Testing
Responsiveness Testing
Testing Techniques Used:
Boundary Value Analysis (BVA)
Manual Testing
Decision Table (for input validation)
Tools & Technologies:
Web Browser (Chrome, Edge)
Browser Developer Tools
Cross-Browser Testing
Key Features Tested:
Contact form validation
Navigation and links
Button functionality
Mobile responsiveness
Conclusion:
The testing process confirmed that the website performs effectively across different scenarios, ensuring reliability, usability, and proper validation of user inputs.
